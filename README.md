## Features

### User Management
- User registration and authentication
- Profile creation (Buyer/Seller modes)
- Email verification
- User ratings and reviews
- Account management

### Marketplace Core
- Product/Service listings
- Multiple categories (Cars, Accessories, Clothes, Services, General Goods)
- Advanced search and filtering
- Image uploads (up to 10 images per listing)
- Listing management (Create, Edit, Delete)
- Featured listings
- Recently viewed items

### Transactions & Payments
- Dual currency support (USD & ZWL)
- Multiple payment methods
- Secure buyer/seller messaging
- Order management
- Transaction history
- Commission structure (transparent fees)

### Communication
- Real-time messaging between buyers and sellers
- Chat history
- Notification system
- Message search

### Admin Dashboard
- User management
- Listing moderation
- Transaction monitoring
- Dispute resolution
- Platform analytics
- Revenue tracking

## Tech Stack

### Frontend
- React.js (Web Dashboard)
- React Native (Mobile - iOS & Android)
- Redux (State Management)
- Axios (API Client)
- Material-UI (Component Library)

### Backend
- Node.js with Express
- MongoDB (Database)
- JWT (Authentication)
- Socket.io (Real-time Messaging)
- Stripe/PayPal Integration (Payments)

### Deployment
- Frontend: Vercel / Netlify
- Backend: Heroku / AWS / DigitalOcean
- Database: MongoDB Atlas
- Mobile: App Store & Google Play

## Installation

### Prerequisites
- Node.js (v14+)
- MongoDB
- npm or yarn

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Configure environment variables
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Mobile Setup
```bash
cd mobile
npm install
npx react-native run-android
# or
npx react-native run-ios
```

## Environment Variables

See `.env.example` files in each directory for required variables.
